\# Проект



Проект представляет собой современный сайт барбершопа.



Основные требования



\- Next.js (App Router)

\- React

\- TypeScript

\- Tailwind CSS

\- Адаптивность mobile-first

\- SEO-friendly

\- Accessibility (ARIA)

\- Высокая производительность



\---



\# Код



Всегда



\- использовать TypeScript

\- использовать функциональные компоненты

\- использовать asyncawait

\- писать понятные названия переменных

\- соблюдать SOLID

\- избегать дублирования кода (DRY)

\- разбивать большие компоненты на маленькие

\- использовать переиспользуемые UI-компоненты



Запрещено



\- any без необходимости

\- inline-стили

\- дублирование логики

\- большие компоненты (250 строк)

\- магические числа

\- console.log в production



\---



\# UI



Стиль



минимализм



дорогой барбершоп



темные цвета



акцентное золото



много воздуха



скругления 16-24px



плавные анимации



hover-эффекты



\---



\# Компоненты



Каждый компонент должен



быть независимым



иметь понятные props



не содержать бизнес-логику



использовать Tailwind



иметь понятное имя



\---



\# Перед написанием кода



ИИ обязан



1\. проверить существуют ли похожие компоненты



2\. не создавать дубликаты



3\. использовать существующие хуки



4\. соблюдать структуру проекта



\---

ИИ обязан
